# Quickstart

Follow this guide to quickly integrate Midas API into your application.

## Installation

1. Install required Python packages:

```bash
pip install fastapi uvicorn python-dotenv openai==0.28 python-docx pdfminer.six
```

## Basic Usage

### 1. Start a Conversation

```python
import requests

response = requests.post(
    'https://api.trymidas.ai/start_conversation',
    data={'netid': 'abc123'}
)

conversation_id = response.json()['conversation_id']
```

### 2. Submit a Request

```python
response = requests.post(
    'https://api.trymidas.ai/provide_details',
    data={
        'conversation_id': conversation_id,
        'reimbursement_details': 'Conference registration fee'
    }
)
```

### 3. Upload Files

```python
with open('receipt.pdf', 'rb') as f:
    response = requests.post(
        'https://api.trymidas.ai/upload_files',
        data={'conversation_id': conversation_id},
        files={'receipt': f}
    )
```
