# Midas API Documentation

The Midas API provides an automated solution for handling reimbursement requests through an AI-powered system.

## Overview

Our API enables you to:
- Process multiple file formats (PDF, DOCX, images)
- Get AI-powered validation
- Receive automated email notifications
- Track reimbursement status
